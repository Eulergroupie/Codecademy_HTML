The Basics of HTML

HTML is the skeleton of all web pages. HTML stands for Hyper Text Markup Language. A markup language is a computer language that defines the structure and presentation of raw text. Hypertext is text displayed on a computer or device that provides access to other text through links, also known as hyperlinks. Learning HTML is the first step in creating websites.

Basic Structure

HTML is composed of elements. These elements structure the webpage and define its content. The diagram below displays an HTML paragraph element. It is made up of an opening tag, the content, and a closing tag. When an element is contained inside another element, it is considered the child of that element. The child element is said to be nested inside of the parent element. Since there can be multiple levels of nesting, this analogy can be extended to grandchildren, great-grandchildren, and beyond. The relationship between elements and their ancestor and descendent elements is known as hierarchy. Understanding HTML hierarchy is important because child elements can inherit behavior and styling from their parent element.

A very basic web page takes this form:

<!DOCTYPE html>
<html>

<header>
<title>This text in the tab</title>
</header>

<body>
<h1>This is a main heading</h1>
<p>This is in a paragraph</p>
<h2>This is a smaller header</h2>
<p>
This is in a paragraph<br>
Line breaks are made with the single tag <br>
if the paragraph doesn't end it
</p>
</body>

<footer>
</footer>

</html>

The first html tags indicate what kind of file this is to a browser that displays it.
The header section generally contains the title and maybe some general information.
The body contains the content of the web page, the basic structure of which is usually composed of headings and paragraphs among other things. Headings are used to title sections and range from large <h1> to small <h6>.
The footer often consists of links to other pages, email adresses, and other general info.

The <div></div> tags are often used to "divide" the page into sections.
If we want to expand an element’s tag, we can do so using an attribute. Attributes are content added to the opening tag of an element and can be used in several different ways, from providing information to changing styling. Attributes are made up of two parts: the name of the attribute, and the value of the attribute. One commonly used attribute is the id. We can use the id attribute to specify different content (such as <div>s) and is really helpful when you use an element more than once. ids have several different purposes in HTML, but for now, we’ll focus on how they can help us identify content on our page.

<div id="intro">
<h1>Introduction</h1>
</div>

<p></p> tags are used to contain blocks (paragraphs) of plain text. <span></span> tags contains short pieces of text or other HTML. They are used to separate small pieces of content that are on the same line as other content. It’s best to use a <span> element when you want to target a specific piece of content with an id that is inline, or on the same line as other text.

Styling Text

The <em></em> tags emphasize text (usually italics). <strong></strong> tags produce bold text.

Lists

An unordered list creates a list with bullets. The <ul></ul> tags surround the entire list, while each line item uses the <li></li> tags like this:

<ul>
<li>Limes</li>
<li>Tortillas</li>
<li>Chicken</li>
</ul>

 to produce:

  • Limes
  • Tortillas
  • Chicken
An ordered list is numbered and is created with the same format except for <ol></ol> tags.

Images

The <img> tag allows you to add an image to a web page. Most elements require both opening and closing tags, but the <img> tag is a self-closing tag. Note that the end of the <img> tag has a forward slash / before the >.
The <img> tag has a required attribute called src. The src attribute must be set to the image's source, or the location of the image. The value is often a uniform resource locator (URL) that indicates the web or local address where the image is stored. For example:

<img src="https://s3.amazonaws.com/codecademy-content/courses/web-101/web101-image_brownbear.jpg" />

 produces...


The alt attribute can be added to the image tag just like the src attribute. The value of alt should be a description of the image.

<img src="file:///C:/Code/sunflowers.jpg" alt="A field of sunflowers" />

 The alt attribute serves the following purposes:

  • If an image fails to load on a web page, a user can mouse over the area originally intended for the image and read a brief description of the image.
  • Visually impaired users often browse the web with the aid of screen reading software. When you include the alt attribute, the screen reading software can read the image’s description out loud to the visually impaired user.
  • The alt attribute also plays a role in Search Engine Optimization (SEO), because search engines cannot “see” the images on websites as they crawl the internet. Having descriptive alt attributes can improve the ranking of your site.

Videos

In addition to images, HTML also supports displaying videos. Like the <img> tag, the <video> tag requires a src attribute with a link to the video source. Unlike the <img> tag however, the <video> element requires an opening and a closing tag.

<video src="myVideo.mp4" width="320" height="240" controls>Video not supported</video>

 After the src attribute, the width and height attributes are used to set the size of the video displayed in the browser. The controls attribute instructs the browser to include basic video controls: pause, play and skip. The text, “Video not supported”, between the opening and closing <video> tags will only be displayed if the browser is unable to load the video.
